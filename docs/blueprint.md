# **App Name**: FleetCostCalc

## Core Features:

- Distance Input: Accepts distance traveled (in km) as numerical input from the user.
- Fuel Consumption Input: Accepts vehicle fuel consumption (km/l) as numerical input from the user.
- Fuel Price Input: Accepts fuel price per liter as numerical input from the user.
- Total Cost Calculation: Calculates the total fuel cost based on the formula: (Distance / Consumption) * Fuel Price.
- Real-time Updates: Automatically updates the total cost whenever any of the input fields change, providing instant feedback to the user.
- Zero Consumption Prevention: Prevents division by zero if the user inputs zero fuel consumption to avoid calculation errors.
- Cost Visualization: Displays the calculated total cost prominently, emphasizing its importance to the user.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey professionalism and reliability.
- Background color: Light gray (#EEEEEE) to provide a clean, neutral backdrop.
- Accent color: Teal (#009688) for interactive elements and to add a touch of modernity.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, providing a modern, objective, neutral aesthetic that is suitable for both headlines and body text.
- Card layout with a subtle shadow and rounded edges to enhance visual appeal.
- Simple and professional icons representing distance, fuel, and cost.
- Subtle transition animations for cost updates, making the app more dynamic and engaging.
